## 2024-05-01 - Form Accessibility Pattern
**Learning:** Native HTML5 validation (`required`, `minLength`) combined with explicit `htmlFor`/`id` linking and `role="alert"` for error messages provides immediate, screen-reader friendly feedback on generic components like `AuthModal` without relying solely on custom JavaScript logic.
**Action:** Apply this pattern consistently across all form implementations to guarantee robust browser-level validation and accessibility.
