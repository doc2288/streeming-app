# Streeming Mobile (Flutter)

Базовий глядач для HLS. Поки без автентифікації; підключається до HLS URL (наприклад, від вашого backend/CDN).

## Запуск
1. Встановіть Flutter 3.3+.
2. `flutter pub get`
3. Запустіть на емуляторі/пристрої: `flutter run`
